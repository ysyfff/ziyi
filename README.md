## idb

- __database__     数据库，里面存放了object store
- __object store__ 类似mysql的table
- __index__        object store的一种，用来组织数据，通过name，age等来获取object store
- __transaction__  事务，一个或多个操作的wrapper，保证数据库的完整性
- __cursor__       遍历多条就的机制
